1. Установить python2
2. Если python не запускается в консоли, добавить в path пути C:\Python27 и C:\Python27\Scripts
3. Запустить install-deps.bat
4. Выполнить
     python avgrabber.py search -q "ps4" -q "playstation 4"
   должен появиться вывод строк в формате csv по запросам ключах -q
   ключей -q пиши сколько нужно, все сработают
